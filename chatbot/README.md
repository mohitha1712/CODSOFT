# Task 1: Rule-Based Chatbot

## Description
This project is a simple rule-based chatbot developed using Python. 
It interacts with users by identifying their intent and responding with predefined messages.

## Features
- Responds to greetings like "hello", "hi"
- Handles user queries with predefined intents
- Gives random responses for better interaction
- Exit option to end conversation

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the program:
   python main.py
